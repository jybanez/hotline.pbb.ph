## Pack Summary

Alpha remains the working reference prototype.

Beta is the new build target and should:
- preserve the Hotline domain
- preserve local-first barangay workflows
- use distinct role-based surfaces
- adopt Helper-first UI
- adopt Realtime for live transport
- postpone command and SITREP to later deliberate phases
- use the contracts doc as the implementation source of truth for DTOs and vocabularies
- use the repo structure, migration plan, and app layer map to reduce kickoff ambiguity
